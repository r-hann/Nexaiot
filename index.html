Platform Features

Semua yang Anda
butuhkan dalam satu
platform

📡 01

Real-time Monitoring

Pantau sensor, aktuator, dan status perangkat secara real-time. Dashboard responsif dengan visualisasi data yang intuitif.

🎛️ 02

Remote Control

Kendalikan perangkat IoT dari mana saja. Toggle relay, atur threshold, kirim perintah langsung ke hardware Anda.

📊 03

Data Analytics

Analisis tren data historis dengan grafik interaktif. Export data, buat laporan, dan buat keputusan berbasis data.

🔔 04

Smart Alerts

Atur notifikasi otomatis berdasarkan kondisi sensor. Email, push notification, atau webhook ke sistem eksternal.

🔌 05

Multi-Protocol

Dukungan MQTT, HTTP REST API, WebSocket. Kompatibel dengan ESP32, Raspberry Pi, Arduino, dan hardware IoT lainnya.

🔒 06

Secure by Design

Enkripsi end-to-end, autentikasi token, dan manajemen akses berbasis peran untuk keamanan data Anda.

How It Works

Mulai dalam
hitungan menit

1
Buat Akun

Daftar gratis, tanpa kartu kredit. Akun langsung aktif dan siap digunakan.

2
Buat Project

Organisir perangkat IoT Anda dalam project. Tambahkan widget sesuai kebutuhan monitoring.

3
Hubungkan Perangkat

Gunakan API key project Anda. Upload kode ke ESP32 / Arduino dan langsung terhubung.

4
Monitor & Kontrol

Data mengalir real-time ke dashboard. Pantau, analisis, dan kendalikan dari mana saja.

nexaiot_device.ino
// NexaIoT ESP32 Example
#include <WiFi.h>
#include <NexaIoT.h>
 
// Konfigurasi
const char* TOKEN = "nx_proj_xxxx";
 
void setup() {
  NexaIoT.begin(TOKEN);
  NexaIoT.connectWiFi("ssid", "pass");
}
 
void loop() {
  float temp = analogRead(A0);
  NexaIoT.send("temperature", temp);
  delay(1000);
}
 
// ✓ Data terkirim ke dashboard anda
